# Smart Cafeteria

This repository contains my final senior-year project in Computer Science. I designed and developed this cafeteria web application as part of my academic work, and I presented it in class in front of the jury.

I am including this project in my college application portfolio as an example of my work in software design, full-stack development, and problem solving.

## Project Overview

Smart Cafeteria is a university cafeteria platform that helps users browse meals, place orders, manage favorites, and interact with an AI assistant.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. (Optional) Create `.env.local` for the contact form:
   `CONTACT_SENDER_EMAIL=you@example.com`
   `CONTACT_RECIPIENT_EMAIL=you@example.com`
   `CONTACT_APP_PASSWORD=your_app_password`
3. Run the app:
   `npm run dev`
